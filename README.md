# Open Campus Advisor — Landing Page

Landing page for [opencampusadvisor.org](https://opencampusadvisor.org).

Built with Next.js + Tailwind CSS, deployed on Vercel.

## Related

- **Core repo:** [open-campus-advisor/open-campus-advisor](https://github.com/open-campus-advisor/open-campus-advisor)
- **REST API:** [api.opencampusadvisor.org](https://api.opencampusadvisor.org)
- **npm:** [open-campus-advisor](https://www.npmjs.com/package/open-campus-advisor)

## Development

```bash
npm install
npm run dev   # http://localhost:3000
```

## Deploy

Auto-deploys to Vercel on every push to `main`.
